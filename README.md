# Создание системы интернет-магазина

## Описание задачи
В процессе выполнения задания необходимо реализовать: 
 Работу с пользователями разных ролей 
 Логику работы с заказами, включая отмену, возврат и обновление 
статуса 
 Реализацию системы заказов с использованием транзакций 
 Применение принципов ООП: полиморфизм, наследование, чисто 
виртуальные функции, управление памятью через умные указатели 
 Адаптивную работу с базой данных через шаблонный класс для 
подключения к PostgreSQL, включая работу с транзакциями, 
хранимыми процедурами, триггерами и функциями 
 Дополнительные функциональные требования, такие как создание 
триггеров для отслеживания изменений статусов заказов и цен 
продуктов
